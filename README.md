# 🌐 Portfolio Web

This project is my personal portfolio, built using modern frontend technologies. Its main purpose is to showcase my skills, experience, and projects in a visually engaging and interactive way.

## 🚀 About the Project

This is not a traditional static portfolio. It is designed as a living project that continuously evolves.

* ✨ It will be regularly updated with new projects, technologies, and experiences.
* 🎨 The design will frequently change as I experiment with new ideas and trends.
* ⚡ It will be continuously improved in terms of performance, accessibility, and user experience.

In short, this repository serves both as a portfolio and as a personal experimentation lab.

## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS
* React Router
* Three.js / React Three Fiber

## 📦 Installation

You can explore the live version of this project here:
🌐 https://hippedtony.github.io/portfolio-dashboard-app/

Clone the repository:

git clone https://github.com/your-username/your-repo.git

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## 🏗️ Build

To generate the production build:

```bash
npm run build
```

## 🌍 Deployment

This project is intended to be deployed on GitHub Pages.
